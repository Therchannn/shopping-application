# Shopping Application

## Frontend
Đã có hỗ trọ thymeleaf để render dữ liệu thông qua các service, đã có phần mẫu ở trên
/resource/static: bao gồm các ảnh, style css và js.
/resource/templates: bao gồm các file HTML.

## Backend
Đã xây dựng các service, đang update thêm
Lưu ý: nhớ chỉnh lại config trong file application.properties về phần db để hoạt động đúng nhé (những phần liên quan đến db thôi nhé).

### Khác
Có gì không rõ thì tự tìm hiểu hoặc hỏi :))
